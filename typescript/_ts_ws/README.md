### Running this sample

1. `temporal server start-dev` to start [Temporal Server](https://github.com/temporalio/cli/#installation).


- start server

``` bash
temporal server start-dev
```

- install dependencies

``` bash
npm install
```

- start WS client

``` bash
npm run ws_client
```


- run the worker

``` bash
npm run start.watch
```


- start workflow

``` bash
npm run workflow
```
